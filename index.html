Projects

Les projets sur lesquels j'ai travaillé, dans le cadre de ma formation Epitech ou bien perso
My Shop

projet Epitech

Site de e-commerce avec son interface d'administration en PHP avec possibilité de s'inscrire et de s'authentifier de manière sécurisée, search bar et filtres.

Free Ads

projet Epitech

Site de petites annonces gratuit avec interface d'administration. Login, register, filter bar, search bar, possiblités d'upload des images et liens de partage.

Post-it

projet Epitech

Interface de sticky notes avec utilisation de l'API fournie par Epitech pour la persistance de données, notes classables par drag and drop.

YOWL

projet Epitech

Clone de Trello avec installation et configuration d'un headless Wordpress afin de pouvoir utilsier son API et faire le lien avec la base de données mySQL.

TWP

projet Epitech

Projet de groupe ayant pour objectif de produire un site de reviews et commentaires avec Vue.js pour le front et Laravel pour le back via une API à développer.

AppMeteo
